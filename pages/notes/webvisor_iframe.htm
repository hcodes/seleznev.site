Для этого из шаблонов сайта необходимо убрать js-код, который препятствует работе сайта внутри IFRAME.

И вставить следующий код:

<script>
try {
    if (self.parent && self.parent !== self) { // Сайт внутри IFRAME?
        if (document.referrer.search(/^[^:]+:\/\/([^.]+\.)?yandex\.(?:ru|ua|net|by|kz|com)/) == -1) {
            self.top.location = document.location;
        }
    }
} catch(e) {}
</script>
